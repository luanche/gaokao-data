# 高考志愿推荐助手（含23-25年历史数据 + 26年招生计划）

根据你的分数（2026年或2025年），自动换算成往年的等效分，再参考 **2023-2025年三年的录取数据** 推荐适合的院校和专业，帮你判断分数涨跌趋势。同时支持查询 **2026年官方招生计划**（物理类/历史类各批次）。

## 核心功能

**2026年分数 → 查一分一段表得到位次 → 用位次找往年等效分 → 参考23-25年数据推荐 → 展示26年计划人数**

## 新增功能：2026年招生计划查询

支持两种模式：

### 模式1：推荐模式（默认）
输入分数后，系统自动推荐冲刺/稳妥/保底院校，**每条推荐结果都附有该院校2026年在重庆该批次的计划招生总人数**，以及各专业2026年的计划人数，帮你判断扩招/缩招趋势。

### 模式2：招生计划查询模式
直接查询2026年官方招生计划，支持：
- **按院校名称搜索**：查看某院校所有专业的招生计划（代码、人数、学制、学费）
- **按专业关键词搜索**：如输入"计算机"，找出所有含该关键词的专业及招生计划
- **列出所有院校**：按计划人数排序显示所有招生院校

## 核心算法：位次换算 + 历年趋势对比

### 换算流程
```
你的2026年分数（如497分）
    ↓ 查2026年一分一段表
得到位次（如64,996）
    ↓ 用位次查往年一分一段表
    2025年≈495分  2024年≈492分  2023年≈451分
    ↓ 用2025年等效分匹配学校
推荐490-500分的院校
```

### 历年趋势对比（精确到每年）

每条推荐都会显示该院校/专业 **2023、2024、2025三年的分数线**，
并且**不是简单地和你的分数比**，而是**和你的排名在各年对应的等效分比**：

```
你的排名64,996在各年等效分：
  2025年≈495分  2024年≈492分  2023年≈451分

四川外国语大学历年：
  2025年:500分 vs 495(你的等效) = ↑5  ✓
  2024年:507分 vs 492(你的等效) = ↑15 ✓
  2023年:462分 vs 451(你的等效) = ↑11 ✓
```

趋势标记含义：
- `↑5` = 比该年等效分高5分（冲刺）
- `↓3` = 比该年等效分低3分（保底）
- `=` = 持平（稳妥）

## 交互流程

### 推荐模式
1. **模式选择** — 推荐模式(1) 或 招生计划查询模式(2)
2. **考试年份** — 2026 或 2025（默认2026）
3. **科类** — 物理类 / 历史类
4. **分数** — 你的高考分数
5. **批次** — 本科批 / 专科批 / 本科提前批B段
6. **浮动范围** — 上下浮动多少分（默认±5）
7. **是否只看985/211** — 可选
8. **偏好省份/地区** — 如重庆、四川等，可选
9. **感兴趣的专业方向** — 如计算机、医学、师范，可选
10. **偏好城市类型** — 一线/新一线/不限制，可选

### 招生计划查询模式
1. **科类** — 物理类 / 历史类
2. **批次** — 本科批 / 专科批 / 本科提前批等
3. **查询方式** — 按院校名称 / 按专业关键词 / 列出所有院校

## 输出内容

- 位次换算结果（2026→2025/2024/2023等效分）
- 冲刺/稳妥/保底院校（每条附23-25年趋势 + **26年计划人数**）
- 各专业推荐（附该专业23-25年历史分数 + **26年计划人数**）
- 2026年招生计划详情（按院校/专业查询）
- 报考建议

## 一键生成交互式志愿规划网页

基于 `combined_table.json`（一表联动，含专业代码、2023-2025三年详细分数、2026招生计划、学校信息等），可以为用户选定的学校和专业生成增强版JSON和交互式HTML网页。

### 数据增强

相比基础版本，增强版每条数据包含：
- **专业代码** — 从 combined_table 中精确匹配
- **专业全称/备注** — 含办学地点、合作院校等详细备注
- **2023-2025三年详细分数** — 最低分、最低位次、平均分、平均位次、最高分、录取人数
- **2026年招生计划** — 计划人数、学制、学费、选科要求、专业层次、门类、专业类
- **学校详细信息** — 所在省、城市、城市水平标签、院校标签、院校水平、隶属单位、类型、公私性质、保研率

### 交互式HTML网页功能

生成的网页支持：
1. **三栏布局** — 冲刺/稳妥/保底分栏展示，颜色区分
2. **拖拽排序** — SortableJS 实现，同栏可拖拽调整优先级
3. **删除/恢复** — 删除到回收区，可恢复
4. **搜索筛选** — 按学校/专业名称搜索，按新增/原有专业筛选
5. **导出志愿表** — 导出 Markdown 格式表格
6. **另存为HTML** — 保存当前排序状态为独立HTML文件
7. **自动保存** — 所有操作保存在浏览器 localStorage

### 运行方式

cd /root/projects/gaokao-data && python3 .pi/skills/gaokao-recommend/scripts/generate_plan.py

输出文件：
- `.tmp/志愿推荐.json` — 增强版JSON数据
