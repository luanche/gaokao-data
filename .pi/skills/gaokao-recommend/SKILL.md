# 高考志愿推荐助手（含23-25年历史数据参考）

根据你的分数（2026年或2025年），自动换算成往年的等效分，再参考 **2023-2025年三年的录取数据** 推荐适合的院校和专业，帮你判断分数涨跌趋势。

## 核心功能

**2026年分数 → 查一分一段表得到位次 → 用位次找往年等效分 → 参考23-25年数据推荐**

## 核心算法：位次换算 + 历年趋势对比

### 换算流程
```
你的2026年分数（如497分）
    ↓ 查2026年一分一段表
得到位次（如64,996）
    ↓ 用位次查往年一分一段表
    2025年≈495分  2024年≈492分  2023年≈451分
    ↓ 用2025年等效分匹配学校
推荐490-500分的院校
```

### 历年趋势对比（精确到每年）

每条推荐都会显示该院校/专业 **2023、2024、2025三年的分数线**，
并且**不是简单地和你的分数比**，而是**和你的排名在各年对应的等效分比**：

```
你的排名64,996在各年等效分：
  2025年≈495分  2024年≈492分  2023年≈451分

四川外国语大学历年：
  2025年:500分 vs 495(你的等效) = ↑5  ✓
  2024年:507分 vs 492(你的等效) = ↑15 ✓（旧算法错算成↑12）
  2023年:462分 vs 451(你的等效) = ↑11 ✓（旧算法错算成↓33）
```

这样你能真正看出：**以你的排名，这所学校往年是更难考还是更容易**。

趋势标记含义：
- `↑5` = 比该年等效分高5分（冲刺）
- `↓3` = 比该年等效分低3分（保底）
- `=` = 持平（稳妥）

## 使用方式

在 pi 中直接告诉我你的分数，我会引导你一步步完成推荐。

也可以直接运行：
```bash
cd /root/projects/gaokao-data && python3 .pi/skills/gaokao-recommend/scripts/recommend.py
```

## 交互流程

1. **考试年份** — 2026 或 2025（默认2026）
2. **科类** — 物理类 / 历史类
3. **分数** — 你的高考分数
4. **批次** — 本科批 / 专科批 / 本科提前批B段
5. **浮动范围** — 上下浮动多少分（默认±5）
6. **是否只看985/211** — 可选
7. **偏好省份/地区** — 如重庆、四川等，可选
8. **感兴趣的专业方向** — 如计算机、医学、师范，可选
9. **偏好城市类型** — 一线/新一线/不限制，可选

## 输出内容

- 位次换算结果（2026→2025/2024/2023等效分）
- 冲刺/稳妥/保底院校（每条附23-25年趋势）
- 各专业推荐（附该专业23-25年历史分数）
- 报考建议

## 数据来源

- `json_data/score_distribution_2026.json` — 2026年一分一段表
- `json_data/latest_school_scores.json` — 2022-2025年院校录取分数
- `json_data/latest_major_scores.json` — 2022-2025年专业录取分数
- `json_data/latest_score_distribution.json` — 2022-2025年一分一段表
